Bienvenido a DocuMed

Sistema integral de gestión médica potenciado por inteligencia artificial. Gestiona pacientes, citas, laboratorios, recetas y más, todo en un mismo lugar con encriptación de datos y sincronización en la nube.

Ver en GitHub Guía de Instalación

Características Principales

Gestión de Pacientes

  • Registro completo de pacientes
  • Búsqueda avanzada
  • Historial médico detallado
  • Edición y eliminación segura

Panel Médico IA

  • Chat con IA médica en tiempo real
  • Análisis de síntomas
  • Diagnósticos inteligentes
  • Recomendaciones personalizadas

Gestión de Citas

  • Programación de citas
  • Calendario integrado
  • Recordatorios automáticos
  • Historial de citas

Laboratorios

  • Órdenes de laboratorio
  • Carga de resultados
  • Visualización de pruebas
  • Seguimiento histórico

Recetas Digitales

  • Emisión de prescripciones
  • Listado de medicamentos
  • Dosificación automática
  • Archivo histórico

Seguridad

  • Encriptación AES-128
  • Autenticación segura
  • Backups automáticos
  • Sincronización en nube

Instalación Rápida

Requisitos

  • Python 3.8 o superior
  • pip (gestor de paquetes)
  • Conexión a internet

Pasos

  1. Clona el repositorio: git clone https://github.com/SantiagoDgz/DocuMed.git
  2. Instala dependencias: pip install -r requirements.txt
  3. Ejecuta: python launcher.py
  4. Accede en: http://localhost:5000
